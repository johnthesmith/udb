# Эмиссионный коэффициент резерва

1. Эмиссионный коэффициент резерва `k_er` отражает реальное текущее соотношение 
между [Оборотом](./circulation.md) и [Резервом](./reserv.md) в [балансе](./balance.md).
2. Определяется как `k_er​∈R+` по формуле:
```
k_er = C / R
```
3. где:
    - `C` — сумма [Оборота](./circulation.md) в [U](./u.md);
    - `R` — сумма [Резерва](./reserv.md) в [U](./u.md).
4. Используется для корректировки распределения эмиссии, чтобы соотношение 
[резерв](./reserv.md)/[оборот](./circulation.md) приближалось к 
[целевому коэффициенту резерва](./k_tr.md).
4. Автоматически выравнивает соотношение [Резерва](./reserv.md) и 
[Оборота](./circulation.md), обеспечивая стабильность системы и прозрачность 
распределения эмиссии, `lim( k_er → k_tr )`.
