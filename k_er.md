# Эмиссионный коэффициент резерва

1. Эмиссионный коэффициент резерва `k_er` отражает реальное текущее соотношение 
между [Оборотом](./circulation) и [Резервом](./reserv) в [балансе](./balance).
2. Определяется как `k_er​∈R+` по формуле:
```
k_er = C / R
```
    где:
        - `C` — сумма [Оборота](./circulation) в [U](./u);
        - `R` — сумма [Резерва](./reserv) в [U](./u).
3. Используется для корректировки распределения эмиссии, чтобы соотношение 
[резерв](./reserv)/[оборот](./circulation) приближалось к 
[целевому коэффициенту резерва](./k_tr).
4. Автоматически выравнивает соотношение [Резерва](./reserv) и 
[Оборота](./circulation), обеспечивая стабильность системы и прозрачность 
распределения эмиссии, `lim (k_er → k_tr)`.
