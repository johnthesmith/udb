# Cумма эмиссионного вознаграждения системы

1. Cумма эмиссионного вознаграждения [UDB](./udb) `u_es` определяется при расчете 
[эмиссии](./emission):

```
    u_eu = u_ec * ( 1 - k_ec )
```
    1. где:
        1. u_ec - [Cумма эмиссии в оборот](./u_ec)
        2. k_ec - [Коэффициент вознаграждения пользователя](./k_ec)
