# Cумма эмиссионного вознаграждения системы

1. Cумма эмиссионного вознаграждения [UDB](./udb.md) `u_es` определяется при расчете 
[эмиссии](./emission.md):

```
    u_eu = u_ec * ( 1 - k_ec .md)
```

    где:
        1. `u_ec` - [Cумма эмиссии в оборот](./u_ec.md)
        2. `k_ec` - [Коэффициент вознаграждения пользователя](./k_ec.md)

2. Выражается в [U](./u.md)
