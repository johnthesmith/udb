# UDB

1. Unit Database — база данных, поддерживающая замкнутое равенство
   `товар = деньги` в своих рамках.
2. Допустимо существование множества UDB по национальным, географическим,
   отраслевым и функциональным критериям.
3. UDB вводит единицу [U](./u.md) для отражения объема учитываемого [товара](./good.md), 
   выполняет ее учет, [эмиссию](./emission.md) и [отзыв](./withdrawal.md).
4. Все [U](./u.md) распределяются между [резервом](./reserve.md) и [оборотом](./circulation.md).
