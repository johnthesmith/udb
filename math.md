# Базовая математическая модель

1. Общий баланс системы поддерживается через конроль [эмисиии](./emission.md), 
осуществляется через эквивалентность денег и [товара](./goods.md) с учетом 
[нормировочного коэффициента эмиссии](./ke.md):

```
    B = Товар * ke
```



2. С другой стороы [Баланс](./balance.md) формируется как сумма резерва и оборота.

```
    B = R + C
```



3. В свою очередь резерв и оборот связаны через [k_et](./k_et) и для идеальной ситуации выражается как:
```
    C = R * k_et
```



4. Для автокоррекции долей [оборота](circulation.md) и [резерва](reserve.md) используется механизм [k_er](./k_er.md):

```
    C = R * k_er
```

при

```
    lim (k_er → k_tr)
```
