# Оборот UDB

1. Оборот `С` - сумма скалярных величн [U](./u.md) на балансе [udb](./udb.md) и 
[пользователей](./user).
0. Показывает фактическое количество (массу) [U](./u.md), которая может быть 
использован для обмена.
0. Учет Оборота выполняет [UDB](./udb.md).
0. Операции с Резервом приватны, за исключением предоставления информации о его объеме.
0. Оборот увеличивается при [эмисии](./emission.md) новых [U](./u.md).
